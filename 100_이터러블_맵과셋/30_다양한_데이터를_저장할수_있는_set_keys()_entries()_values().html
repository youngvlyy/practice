Set 생성자 함수를 이용한 객체

        1. Set은 중복을 허용하지 않는 값을 모아놓은 특별한 자료구조
        2. Set은 Key가 필요없으며, value만을 저장한다.
        3. Set은 수학적 집합을 구현하기 위한 자료구조이다.

        4. Set에는 다음과 같은 메소드와 프로퍼티가 있다.

            const mySet = new Set();

            mySet.add(값) - 데이터 추가
            mySet.delete(값) - 데이터 삭제
            mySet.has(값) - 데이터 존재여부 확인 true/false
            mySet.clear() - 모든 요소 삭제(완전히 비우기)

            mySet.size - 저장된 데이터 개수
