Map 생성자 함수를 이용한 객체

        1. 지금까지 데이터를 저장하는 방법은 - 일반변수, 배열변수, 객체변수

        2. 현실세계 반영하기엔 턱없이 부족해서 Map, Set 등장

        3. Map은 key가 있는 데이터를 저장한다는 점에서 Object와 유사

        4. 다만 Map은 size 속성으로 항목의 수를 할 수 있게 한다는 점에서 차이가 있다.

        5. Map에는 다음과 같은 메소드와 프로퍼티가 있다.
            const myMap = new Map();

            myMap.set(key, value) - key를 이용해서 value를 저장
            myMap.get(key) - key에 해당하는 값을 리턴, 없으면 undefined
            myMap.delete(key) - key에 해당하는 값을 삭제(메소드체인 불가);
            myMap.has(key) - 존재여부 확인 --> true/false;
            myMap.clear - 완전히 비우기;
            myMap.size - 요소의 개수를 리턴
