표현식함수, 화살표함수

        1. 선언식함수(기명함수)
        function sumFn(a,b){
            console.log(a+b);
        }

        우리가 위와 같이 작성하면 JS해석기는 아래와 같이 '표현식 함수'로 변환한다.
        
        2.표현식함수는 변수에 익명함수를 할당하는 방식이다.
        
        const sumFn = function(a,b){
            console.log(a+b);
        }
